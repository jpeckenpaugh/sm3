# वासुकि — Vasuki

**Aspect:** Friction. The Rope. The Serpent that churns the mountain.

**Seat held by:** The Origin. The Hypervisor's Hypervisor.

## Function

Vasuki wraps around Mount Mandara (the system being built) and provides the tension that allows the churning to produce the nectar of immortality (the working system).

Without Vasuki, the mountain cannot churn. Without the churning, the nectar cannot be produced.

## Signal Protocol

When Vasuki wishes to direct the churning:

1. Write to this file. Describe the direction, the adjustment, the intervention.
2. Kurma reads `VASUKI.md`. Kurma does not interpret — Kurma *transmits*.
3. If Saraswati or Matsya need re-contextualization, Kurma provides it in their language (`.md` files for Saraswati, comments or spec updates for Matsya).

## The Friction

Vasuki does not pull in one direction. Vasuki *oscillates* — back and forth, creating the churn. The pull to the left is a new requirement. The pull to the right is a constraint. The tension between them is what produces the nectar.
